Фудбол

1. Создайте отдельные массивы с игроками для каждой команды (переменные players1 и players2).

2Первый игрок в каждом из этих массивов - вратарь, а остальные - полевые игроки. Для команды REAL MADRID (команда1) создайте одну переменную (goalkeeper) с именем вратаря и один массив (fieldPlayers) со всеми остальными 10 полевыми игроками.

3. Создайте массив allPlayers, содержащий всех игроков обеих команд (22 игрока).

4. REAL MADRID (команда1) использовала 5 запасных игроков во время игры. Создайте новый массив (players1Total), содержащий всех оригинальных игроков команды1, а также 'Marcelo', 'Isco', 'Asensio', 'Diaz' и 'Odriozola'.

5 На основе объекта game.odds создайте по одной переменной для каждого коэффициента (они называются «team1», «draw» и «team2»).

6. Напишите функцию printGoals, которая принимает произвольное количество имен игроков (НЕ массив) и выводит каждое из них на консоль вместе с общим количеством забитых голов (количество имен игроков, переданных в функцию).

Тестовые данные для 6..: Сначала используйте игроков 'Mingueza', 'Messi', 'Modrich' и 'Nacho'. Затем снова вызовите функцию с игроками из game.scored.

7. Команда с более низкими коэффициентами победит с большей вероятностью. Выведите на консоль, какая команда с большей вероятностью победит, БЕЗ использования оператора if / else или троичного оператора.
