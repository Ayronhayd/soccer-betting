Фудбол Продолжение

1. Пройдитесь по массиву game.scored и выведите на консоль имя каждого игрока вместе с номером гола (пример: «Гол 1 - Кроос»).

2. С помощью цикла вычислите средний коэффициент и выведите его на консоль.

3. Выведите в консоль 3 коэффициента, но отформатированные следующим образом: Коэффициент на победу РЕАЛ МАДРИД: 1.48 Коэффициент на ничью: 2.53 Коэффициент на победу «БАРСЕЛОНЫ»: 4.25 Получайте названия команд непосредственно из игрового объекта, не кодируйте их жестко.

4. Бонус: создайте объект под названием goalScorers, который содержит имена игроков, забивших голы, в качестве свойств и количество голов в качестве значения. В этой игре он будет выглядеть следующим образом: { Kroos: 1, Benzema: 1, Mingueza: 1 }
